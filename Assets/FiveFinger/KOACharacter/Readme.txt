A unit used in serviced games.


1 kinds of character (6 kinds of skins by grade, 6 kinds of monster skins by each grade)

- walking animation
- Attack animation
- Skill animation

How to use

Select and play the Scene file at Fivefinger / KOACharacter / CharacterScene


Testing completed in Unity 5.6.6f2.

Testing completed at spine runtime spine-unity-3_6-2018-01-09.

Spine Download: https://github.com/EsotericSoftware/spine-runtimes/tree/3.6/spine-unity
